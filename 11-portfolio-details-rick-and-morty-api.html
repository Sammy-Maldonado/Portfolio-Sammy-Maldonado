API de Rick and Morty

Desarrollo de una wiki de Rick and Morty utilizando el framework React Js, la herramienta de desarrollo Rect Vite y el consumo de la "API de Rick and Morty".

Se ponen a prueba principalmente las siguientes herramientas/conceptos:
- React Hooks[useState, useEffect]
- Fetch API
- Bootstrap
- Paginación
- Filtrado de datos
- Barra de busqueda
- React Router
- Ruteo dinámico
